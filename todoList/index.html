<todo-list>

<todo-list> is an web components demo. It's a great introduction to building web components with LitElement, with nice documentation site as well.

As easy as HTML

<todo-list> is just an HTML element. You can it anywhere you can use HTML!

<todo-list></todo-list>

Configure with attributes

<todo-list> can be configured with attributed in plain HTML.

<todo-list title="a new title"></todo-list>

Declarative rendering

<todo-list> can be used with declarative rendering libraries like Angular, React, Vue, and lit-html

import {html, render} from 'lit';

const title = 'In diffrent library';

render(
  html`
    <h2>This is a &lt;todo-list&gt;</h2>
    <todo-list .title=${title}></todo-list>
  `,
  document.body
);
